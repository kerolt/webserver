顶层const与底层const

发表于 2023-12-26 分类于 C++

简记·

靠右的const是顶层const，靠左的const是底层const

顶层const·

即指针本身是个常量

出错原因是因为pa1是一个顶层const，即其只能指向a，无法修改其指向b。

底层const·

即指针所指的的对象是个常量

pa2是一个底层const，即pa2所指对象的值不可改变，但是pa2所指向的对象可以改变

注意：用于声明引用的const都是底层const

【链表】判断回文链表

发表于 2023-11-10 分类于 算法

https://leetcode.cn/problems/palindrome-linked-list/

（1）将链表转化为数组进行比较·

比较呆板的做法，空间复杂度为O(n)​。

（2）递归·

链表也具有递归性质，二叉树也不过是链表的衍生。

利用后序遍历的思想：

先保存头结点（left，全局变量），然后递归至最后（最深）的结点（right），然后比较left​和right​的值；如果相等，由递归栈返回上一层（也即right向左走），再操作left向右走，这样就实现了left和right的双向奔赴。

（3）优化递归·

利用方法二，看似是没有使用到额外空间了，但实际上还有递归所带来的函数调用栈的开销，其空间复杂度也为O(n)​。

因此可以利用双指针的思想，找到链表的中间结点后，将其后面的结点反转。
